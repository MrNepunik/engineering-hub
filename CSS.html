01 The Founder

Håkon Wium Lie

In 1994 at CERN, Lie proposed Cascading Style Sheets. His core theory: design should be separate from content. Before CSS, changing the color of 100 pages meant editing every one. Lie's "Cascading" theory allows styles to flow from one source to many.

02 The Critical Rendering Path

When a browser loads CSS, it follows a mathematical process:

  1. DOM + CSSOM: HTML (DOM) and CSS (CSSOM) combine into a Render Tree.
  2. Layout: Calculates exact geometry (width, height, position) of every box.
  3. Paint: Fills in pixels — colors, shadows, images.
03 Specificity Math

When multiple rules target the same element, CSS uses a point system:

  • Inline Styles: 1000 points
  • IDs (#header): 100 points
  • Classes (.btn): 10 points
  • Elements (h1): 1 point

Example: #nav .link = 110 points. Always beats .link = 10 points.
